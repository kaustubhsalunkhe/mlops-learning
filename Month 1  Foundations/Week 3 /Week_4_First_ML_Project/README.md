# First ML Project – Purchase Prediction

## Problem Statement
Predict whether a user will buy a product based on age and salary.

## Dataset
- age
- salary
- buy (0 = No, 1 = Yes)

## Project Structure
- `train.py` – trains and saves the model
- `predict.py` – loads model and makes predictions
